# Simplified Works — Website

A corporate, multi-page website for **Simplified Works**, a digital solutions agency
specializing in website development, digital branding, and custom web solutions for
businesses, schools, startups, salons, restaurants, and clinics.

## Pages
| File | Purpose |
|------|---------|
| `index.html` | Home — hero, stats, services overview, about teaser, industries, process, testimonials, CTA |
| `services.html` | Detailed services + add-ons |
| `about.html` | Story, mission, vision, values, industries |
| `contact.html` | Contact details + quote request form |

## Shared assets
- `styles.css` — full corporate stylesheet (navy / teal / gold palette, responsive)
- `script.js` — mobile menu, animated counters, contact form handling
- `assets/` — generated logo, hero, and about images

## How to use
Open `index.html` in any browser. To publish, upload the entire `simplified-works`
folder to any static host (Netlify, GitHub Pages, Vercel, cPanel, etc.).

## Notes / next steps
- The contact form is front-end only. To receive submissions, connect it to a backend
  or a form service (e.g. Formspree, Getform) or `mailto:`.
- Replace the placeholder testimonials and stats with real ones when available.
- Update the social links in the footer if you add Instagram/Facebook/LinkedIn.
